# IMAGE_MANAGEMENT
基于Vue3+Element Plus+Django的本地图片管理工具，包括图片的查看、分类、移动、复制、删除、重命名、元数据修改、固定命名格式批量重命名、按照日期查看等功能

## 开发服务器启动
### 前端
`cd fontend`

`npm install`

`num run dev`
### 后端
 - 最好使用虚拟环境

`pip install -r requirements.txt`

`cd backend`

`python manage.py runserver`
